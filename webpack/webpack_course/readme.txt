Переменные окружения

Можно использовать любой формат переменных окружения:
- передать их при запуске в консоли
- создать .env file
- или задать их на уровне операционной системы

Еще одна схема (package.json)
"build:dev": "webpack --env mode=development",
"build:prod": "webpack --env mode=production"

Вот так с помощью переменных окружения можно передавать какую-то конфигурацию извне,
чуть позже мы будем передавать порт, еще какие-то опции

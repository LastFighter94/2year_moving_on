LARAVEL это model view controller FRAMEWORK

Model - дата которую мы можем представлять с помощью любой СУБД
View - то что пользователь видит в HTML, template короче (blade шаблонизатор)
Controller (router) - позволяет пользователю безопасно взаимодействовать с моделью

Artisan - command line tool to create our model

ORM - object relational mapping

(In project folder - terminal)

php artisan make:migration create_listings_table 

(After editing the create_listings_table) php artisan migrate  

=> then in mysql brad user (use laragigs, SHOW TABLES;)



(После раскомментирования databaseSeeder.php)

php artisan db:seed 


php artisan migrate:refresh (убираем пользователей обратно)
php artisan migrate:refresh --seed (обвонляем и seed)



php artisan make:model Listing





Optional (create factory)
php artisan make:factory ListingFactory





php artisan make:controller ListingController

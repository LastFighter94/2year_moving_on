SSR
Server side rendering
1) идет запрос на сервер
2) сервер отправляет в ответ уже отрендеренную страницу (рендер идет на стороне сервера)
3) потом приходит JS bundle - который инициирует процесс под названием Hydration - чтобы сделать доступной всю функциональность vue для контента страницы
(Подробнее о hydration)
https://www.youtube.com/watch?v=TjYLsIxiErU&list=PLyeqauxei6jfFxaO94jsBBtsSPWaBSPSZ&index=1

CSR
Client side rendering
1) идет запрос на сервер
2) сервер отрпвляет в ответ пустую html страницу
3) потом сервер отправляет JS файл, который делает рендер страницы (на стороне браузера)
4) Приложение рендериться
--- боты по индексации поисковиков (crawlers) тупят - также может тормозить все если девайс пользователя слабоват

UNIVERSAL RENDERING
SSR при запросе
CSR при SPA роуминге внутри страницы

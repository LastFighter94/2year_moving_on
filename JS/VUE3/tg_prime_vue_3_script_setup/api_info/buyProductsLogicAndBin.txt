Есть отдельный маршрут, который прописываем в ../src/router/index.js

Речь идет о маршруте в котором аккуратно и понятно расписан весь перечень товаров. Каждый
из товаров можно посмотреть, также к нему можно посмотреть дополнительную информацию (ссылка на youtube, отзывы, примеры использования и так далее).

Каждый товар можно пометить "на покупку".

При пометке на покупку в сущности "Корзина" наполняются данные, которые потом, с помощью end-points на backend - мы будем отправлять на сервер.
Отправка этих данных происходит на отдельном маршруте, который мы тоже прописываем в ../src/router/index.js

Там, в маршруте корзины - есть две следующие функции

- убрать товар из корзины
- оплатить все товары
- ввести какие-то промокоды и так далее, перед оплатой всех товаров

При покупке товаров происходит запись в историю покупок по nickName пользователя
Очистка корзины
